# StudyTools
Study with free flashcards. I saw a need for this, so I created it with the help of claudeAI for myself and friends. 

[https://dibin17.github.io/StudyTools/](https://dibin17.github.io/StudyTools/)

* Add projects, in each project you can add unlimited flashcards.
* Quiz yourself with a written quiz, or a normal quiz.
* Add a timer for maximum learning.
* Import and export projects, so you can share projects with friends.


That's it. For free.
